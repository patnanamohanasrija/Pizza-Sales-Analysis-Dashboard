# Pizza Sales Analysis Dashboard (Power BI)
This project showcases an end-to-end sales analysis solution built in Power BI using a pizza sales dataset. The dashboard is designed to transform raw transactional data into meaningful business insights through interactive visualizations and dynamic KPIs. It enables users to analyze revenue trends, customer ordering behavior, and product performance across different categories, sizes, and time periods. By highlighting best and worst-selling items, identifying peak sales periods, and uncovering patterns in customer preferences, this dashboard supports data-driven decision-making and provides a clear understanding of overall business performance.
